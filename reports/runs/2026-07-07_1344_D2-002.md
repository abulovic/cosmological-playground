## Result

The Meaning Alignment Institute (MAI) "first moral graph" is imported as sourced
reference data: **59 value cards** and **71 wiser-than edges** from the study of
~500 US participants on three prompts. The two objects are normalized into two
**separate** JSONL files — the firewall made physical: card text lives only in
the values file, and the human-voted ranking lives only in the edges file, so
neither can be reconstructed from the other. A provenance manifest records the
source, revision (sha256), unresolved license, row counts, and schema.

We imported the **published data** (the `dft` JSON export), not the
`moral-graph-elicitation` tool — the tool generates new graphs and needs OpenAI +
Postgres, whereas the current tasks need a fixed, sourced value vocabulary to
match text against.
