# Meaning Alignment Institute values integration

This directory will contain source manifests and normalized MAI value-card records.

Two objects must remain distinct:

1. **Value cards / attention policies:** may be retrieved by semantic relevance.
2. **Moral-graph edges:** contextual judgments that one value is wiser or more comprehensive than another.

The application must not infer the second from the first. Before importing data, record repository URL, revision, license or unresolved license status, row count, schema, and import date in `manifest.json`.
