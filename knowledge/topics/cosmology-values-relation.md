## The relation

Cosmology and values stand in an *elective affinity* (Weber's *Wahlverwandtschaft*),
not an entailment. A cosmology is a "model of" the world that doubles as a "model
for" living (Geertz), so holding it makes some values *easier to hold*. It does not
entail action: by Hume's is/ought gap, no "ought" follows from an "is" by logic
alone. The link runs in three arrows — **down** (cosmology → values: an affordance,
not a derivation), the **affinity itself** (a mutual fit; each makes the other feel
natural), and **up** (need → cosmology: people revise the picture to meet inner need —
James's "will to believe"; MacIntyre's tradition as "an argument extended through
time"). In one line: the cosmology makes some values easy to hold, and need pushes
back on the cosmology.

## Working definition of cosmology

A background picture of what is real and how it hangs together — here, claims across
the twelve frozen primitives. Three features: it is an "is," not an "ought" (it
describes reality and thereby makes some ways of living look fitting, but does not
contain values as premises — the firewall); it is held as mostly-tacit background
(Taylor's "social imaginary"); and it is revisable under need.

## What remains when the cosmology is removed

The values do not vanish; they become **orphaned** — still held, but no longer *made
easy* by a picture of reality (see the `cosmological-orphaning` concept note). What is
left is (1) the standing human situation — the perennial questions any cosmology must
answer, and the needs that pull on belief (James) — and (2) the values themselves, now
unhosted, holdable only by effort (Weber's "disenchantment"; Taylor's "malaise"). So
the cosmology's role was to make certain values *easy*; remove it and the values are
still there but hard to hold. This makes disenchantment a design space: different
cosmologies re-host different subsets of the perennial values.

## The thinkers

- **Hume** — is/ought gap: why the link is not entailment.
- **Geertz** — "model of / model for": how a picture of reality doubles as a guide to
  living.
- **Weber** — "elective affinity": the mutual fit that *is* the relation;
  "disenchantment": what losing the cosmology feels like.
- **Taylor** — "framework / social imaginary": the tacit background that makes some
  goods reachable.
- **James** — "will to believe": belief answering to need (the return arrow).
- **MacIntyre** — tradition as "an argument extended through time": revision from
  within (the Aquinas case).
- Adjacent: **Bourdieu** (habitus — background as felt inclination); **Mary Douglas**
  (grid/group — a typology of cosmology-and-society co-production).

Anchor: Weber × Geertz, inside Hume, with James / MacIntyre for the return arrow.

## Design consequence

The tool must not output "cosmology ⇒ do X." For a held cosmology it surfaces which
perennial values it makes *easier* to hold (re-hosts), which it makes *harder*
(orphans), which questions it opens, and the branch points where different moves
become live. Matching (D4-002) reports "makes value X easier to hold," never "entails
action Y." Output is a map of affordances and branches, not a verdict.
