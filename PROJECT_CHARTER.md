# Project charter

## Goal

Create an inspectable web application that accepts text, one webpage, PDF, or DOCX and returns:

1. cosmological primitives supported by exact passages;
2. ranked relationships to scoped historical or contemporary cosmologies;
3. candidate agreements, contradictions, and missing evidence;
4. implied values mapped to real MAI value cards when available;
5. separate evidence coverage, candidate fit, and calibrated confidence;
6. an explicit hybrid, novel, or insufficient-evidence result when appropriate.

## Seven-day boundary

The deliverable is a bounded research prototype. It does not include recursive crawling, authoritative classification of people, universal cultural coverage, or production-grade calibration.

## Success criteria

- A user completes the supported input-to-report flow.
- Every material inference links to a submitted-text span.
- Every reference claim links to provenance.
- The system distinguishes endorsement from quotation, criticism, reporting, and fictional speech.
- At least 24 scoped cosmology cards are available.
- Value-card similarity and moral-graph position are never conflated.
- Evaluation includes canonical, mixed, sparse, adversarial, and out-of-distribution cases.
- The interface makes uncertainty and limitations legible.

## Human gates

- Domain review of the primitive ontology and cosmology cards.
- Licensing confirmation for imported datasets.
- Human labels for the calibration split.
- Privacy and public-deployment approval.
